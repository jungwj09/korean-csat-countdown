# 수능보기까지 남은 시간
> 08년생부터 15년샹까지, 수능보기까지 얼마나 남았는지 확인할 수 있는 카운트다운입니다.

수험생들이 공부할 때 아이패드나 핸드폰에 띄워두고, 줄어드는 시간을 보며 긴장감과 동기를 동시에 얻을 수 있도록 만든 수능 카운트다운 웹사이트입니다. <br />
밀리초 단위로 실시간 감소하는 타이머로 지금 이 순간의 급박함을 느낄 수 있고, 주기적으로 바뀌는 응원 메시지로 공부 의지를 이어갈 수 있습니다.

## Tech Stack

### Framework
- React 19 + TypeScript + Vite

### Deployment
- Cloudflare Pages
